# ssho-dev-wiki
> 스쇼 개발 위키

## 프로젝트

 - ssho-core-api
   - 코어 API(Spring Boot)
 - ssho-log-api
   - 로그 API(Spring Boot)
 - ssho-reco-api
   - 추천 API(Flask)
 - ssho-prototype-ver2
   - 클라이언트(React Native)
 - ssho-data-analysis-server
   - 데이터 분석 및 집계 백엔드(Node.js)
 - ssho-admin-web
   - 어드민 웹(React)
 - ssho-item-crawler
   - 상품 크롤러(Puppeteer)
